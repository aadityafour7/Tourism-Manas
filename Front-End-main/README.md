# Front-End

# Commands To Setup:-
   1.  git init
   2.  git remote add origin https://github.com/MiniProject-Tours/Front-End.git
   3.  git pull origin main

# Commands to Pull:-
   1. git pull origin <branch-name>

# Command to push:-
   1. git push -u origin <branch-name>

# Sequence to Do:-

    Pull ->   Make changes   ->   Push
